# CodeAlpha Backend Development Internship Tasks
This repository contains all three major tasks completed during my Backend Development Internship at CodeAlpha. All projects are built using **Node.js** and **Express.js**.
**Tasks Overview:**
** 1. URL Shortener **
* **Folder Name:** `CodeAlpha_URL_Shortener`
* **Description:** A web application that takes a long URL and generates a short, redirection-friendly link. It includes dynamic analytics/click tracking.
* **Default Port:** `5000`

** 2. Blog Content Management System (CMS)**
* **Folder Name:** `codelalpha_blog-cms`
* **Description:** A complete CRUD-based content management system where users can create, view, read full articles, and delete blog posts dynamically.
* **Default Port:** `4000`

** 3. Expense Tracker API**
* **Folder Name:** `CodeAlpha_Expense_Tracker`
* **Description:** A personal finance management application that lets users log daily expenses, categorize them, specify dates, and view a dynamically updated total balance.
* **Default Port:** `8000`
